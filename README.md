# Bachelor

A very cool compiler for a very cool programming language.

## Installation

1. Download the repository
2. `cd` into the directory
3. Run CMake: `cmake .`
4. Run make: `make`

## Usage

Run `prog.out`. If you are already in the directory, you can run `./prog.out` to do this.

## Testing

1. `cmake .`
2. `make`
3. `ctest`
